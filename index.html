========================================================= EVORA AI BUSINESS INTELLIGENCE ========================================================= Sekarang upgrade seluruh sistem AI menjadi Business Intelligence Engine. AI bukan hanya menampilkan angka. AI harus mampu: menganalisis memprediksi menjelaskan memberikan rekomendasi memberikan alasan memberikan confidence memberikan tindakan berikutnya. ========================================================= SINGLE SOURCE OF TRUTH ========================================================= Semua modul wajib menggunakan satu sumber data. Target Bulanan Forecast Dashboard Campaign AI Progress Grafik Laporan harus membaca data yang sama. JANGAN menghitung target di banyak tempat. Gunakan satu object pusat. Contoh currentMonthlyTarget Semua halaman membaca object tersebut. ========================================================= MODE TARGET ========================================================= Tambahkan dua mode. Mode AI Mode Manual Default: Mode AI Jika Mode Manual aktif user dapat menentukan target sendiri. Saat aplikasi dibuka kembali target tetap tersimpan. ========================================================= AI HARUS MEMBACA DATA LAMA ========================================================= AI tidak boleh hanya membaca hari ini.
